# shell-aliases

node files to create shell aliases

## Usage

### All shortcuts

```bash
SHELL_ALIAS_DIR=$PATH_TO_REPOSITORY
source $SHELL_ALIAS_DIR/.source.sh
```

### Specific shortcuts

```bash
SHELL_ALIAS_DIR=$PATH_TO_REPOSITORY
alias myAliasName='node $SHELL_ALIAS_DIR/scripts/my-file-to-run.mjs'
```
